Instructions to Add a New Variable:

  • After completing the relevent python function. Open the app.py file
  • After line 12 add the following code to add a variable:
    new_variable = float(request.form["new_variable"])
  • Modify the code in line number 17 to include your new variable. For example:
    result = diabetes_predictor(glucoseConcentration, serumInsulin, BMI, diabetesPedgreeFunction, new_variable)
  • After line 208 add the following code:
    <label for="new_variable">Please enter your new_variable:</label>
<input type="text" name="new_variable" id="new_variable" placeholder="new_variable" required>
  • Your new variable has been added! Modify the code as per your requirement.
